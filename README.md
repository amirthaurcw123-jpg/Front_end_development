# 🎯 Interactive Quiz Application

## 📌 Project Overview

The Interactive Quiz Application is a simple web-based quiz system developed using HTML, CSS, and JavaScript. It allows users to register/login, create new quiz questions, participate in quizzes, and view their final score. The application stores quiz data and user data using the browser's Local Storage.

---

## 🚀 Features

- 👤 User Registration and Login
- 📝 Create Custom Quiz Questions
- 📚 Preloaded Quiz Questions
- ▶️ Start Quiz
- ✅ Multiple Choice Questions
- 📊 Automatic Score Calculation
- 🏆 Result Page
- 💾 Data stored using Local Storage
- 🎨 Responsive and User-Friendly Interface

## ⚙️ How to Run the Project

1. Download or Clone the repository.
2. Open the project in Visual Studio Code.
3. Install the Live Server extension.
4. Right-click on **index.html**.
5. Select **Open with Live Server**.
6. Register/Login.
7. Create your own quiz or start the quiz directly.
8. View your final score on the Result page.

---

## 📖 Project Modules

### 1. Home Page
- Displays the welcome screen.
- Navigation to Login, Create Quiz, and Start Quiz.

### 2. Login Module
- User Registration
- User Login
- Stores user details using Local Storage.

### 3. Create Quiz Module
- Add new quiz questions.
- Four options for each question.
- Select the correct answer.
- Questions are saved in Local Storage.

### 4. Quiz Module
- Displays one question at a time.
- User selects an answer.
- Score is calculated automatically.

### 5. Result Module
- Displays the total score.
- Shows number of correct answers.
